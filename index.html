Quick start up guide

Choose a gamemode and fill in the fields in the selection menu. Sound and Dark mode options will be saved in the localStorage of your browser. Press the start button and confirm to start the countdown. Use the spacebar to change the current player. You may use the pause button to pause the game or the stop button to stop it. After the game ends, press the end button in order to go to the analysis menu, from where you can view per move time. Enjoy!
